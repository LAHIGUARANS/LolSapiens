# LolSapiens

Generate a build for a champion using LoLalytics data.

## Install it from PyPI

```bash
pip install lolsapiens
```

## Usage

Edit input.env:

```bash
$ python -m project_name
#or
$ project_name
```

## Development

```bash
git clone https://github.com/jspenaq/LolSapiens.git
cd LolSapiens
pip install -r requirements.txt
```
